Citta is an AI-powered emotional companion platform available on web (Next.js) and mobile (React Native/Expo), offering customizable, emotionally intelligent AI companions with personalized avatars, voices, traits, and multilingual support tailored for Indian users. Designed as a compassionate companion rather than a standard chatbot, it provides supportive conversations, voice interaction, memory-based personalization, wellbeing encouragement, and daily emotional check-ins. The platform operates on a subscription model with a 72-hour free trial, INR-based pricing, secure Stripe payments, and strong privacy/security compliance aligned with Indian regulations, ensuring safe, consent-driven, and highly personalized user experiences.
