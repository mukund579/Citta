**Citta – AI Emotional Companion Platform**
Citta is a cross-platform AI-powered emotional companion designed to provide meaningful, supportive, and personalized conversations through web and mobile experiences.

**Vision**
•	Modern web app built with Next.js
•	Mobile app powered by React Native + Expo
•	Emotionally intelligent, gender-customizable AI companions
•	Realistic avatar and personality customization
•	Subscription-based model with a 72-hour free trial

**Core Features**
•	Personalized AI companion creation:
•	Gender, voice, tone
•	Avatar appearance
•	Personality and emotional traits
•	High IQ + high EQ conversations
•	Emotional support, daily check-ins, and encouragement
•	Voice-enabled communication
•	Multi-language support for Indian users
•	Long-term memory and personalization

**Web Application**
•	Tech Stack
•	Frontend: Next.js 14, React, TypeScript, Tailwind CSS
•	Backend: Node.js, Express, TypeScript
•	Database: PostgreSQL
•	Payments: Stripe
•	AI: OpenAI GPT API or equivalent
•	Localization: next-i18next / react-i18next
